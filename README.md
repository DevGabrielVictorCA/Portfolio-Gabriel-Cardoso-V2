# 💼 Portfólio - Gabriel Cardoso

Portfólio desenvolvido com foco em UI/UX, responsividade e apresentação estratégica de projetos.

O projeto foi pensado para demonstrar não apenas habilidades técnicas em frontend, mas também a capacidade de estruturar interfaces claras, modernas e orientadas à experiência do usuário.

## 🎯 Objetivo

Criar um portfólio moderno e funcional que:

- Apresente projetos de forma clara e atrativa
- Destaque habilidades em desenvolvimento frontend e UI/UX
- Proporcione uma experiência consistente em diferentes dispositivos
- Sirva como ferramenta profissional para oportunidades na área

---

## 🧠 Conceito e Abordagem

O projeto foi desenvolvido utilizando abordagem **mobile-first**, garantindo que a experiência em dispositivos móveis fosse prioridade desde o início.

O design foi baseado em:

- Hierarquia visual bem definida  
- Uso estratégico de contraste  
- Organização de conteúdo para escaneabilidade  
- Interface limpa e moderna  

---

## 🧩 Funcionalidades

- ✅ Layout responsivo (mobile-first)
- ✅ Seção de projetos com destaque visual
- ✅ **Modais interativos para cada projeto (principal diferencial)**
- ✅ Navegação fluida entre seções
- ✅ Animações suaves com AOS (Animate On Scroll)
- ✅ Botão de contato direto via WhatsApp

## 🧠 Destaque do Projeto

### 🔥 Modais de Projetos

O principal diferencial deste portfólio é a apresentação dos projetos através de modais interativos, que funcionam como mini estudos de caso.

Cada modal apresenta:

- Contexto do projeto  
- Problema identificado  
- Solução aplicada  
- Resultado final  
- Tecnologias utilizadas  

Essa abordagem melhora a experiência do usuário e valoriza a apresentação dos projetos.
